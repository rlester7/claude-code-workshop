Installation Step 1

Open Terminal. Paste this.

You'll run this same command 2-3 times. That's expected!

Troubleshooting

If Claude doesn't open after installing...

Run this command to fix your PATH:

echo 'export PATH="$HOME/.local/bin:$PATH"' >> ~/.zshrc && source ~/.zshrc

Then type claude to start!
